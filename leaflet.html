About this Leaflet map

This map loads bike path data directly from the SBCTA ArcGIS REST service. Segments are symbolized by facility class (I, II, III) and include pop-ups describing status, city, and length in miles. This serves as the non-Esri web mapping component for the GIS 661 final project.
